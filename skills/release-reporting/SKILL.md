# Release Reporting Skill

Use at the end of an onboarding or quantization workflow.

## Report Contents

- recipe ID and version,
- artifact ID and stage,
- compute pool and runtime,
- eval metrics,
- benchmark metrics,
- device-farm KPI matrix,
- failed KPIs and root-cause hypotheses,
- recipe feedback/revision IDs,
- approval and promotion status.

## MCP Tools

- `generate_onboarding_report`
- `get_artifact`
- `promote_artifact`
